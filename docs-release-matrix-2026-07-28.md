# Documentation release matrix — MCP 2026-07-28

This matrix tracks the documentation work remaining for the 2026-07-28 spec release. Scope: the full spec delta from 2025-06-18 to the current draft, including the intervening 2025-11-25 release. `blog/` and the generated SEP pages (`docs/community/seps/`, `docs/seps/` are regenerated from `seps/`) are excluded. Generated on 2026-07-06.

## Structural page moves

The draft spec restructures several pages relative to 2025-11-25:

- **Deleted:** `basic/lifecycle.mdx`, `basic/utilities/ping.mdx`
- **Split:** `basic/transports.mdx` into `basic/transports/{index,stdio,streamable-http}.mdx`
- **Added:** `basic/patterns.mdx`, `basic/versioning.mdx`, `basic/utilities/mrtr.mdx`, `basic/utilities/subscriptions.mdx`, `server/discover.mdx`, `server/utilities/caching.mdx`, `deprecated.mdx`

Follow-ups needed in `docs/docs.json`: when 2026-07-28 becomes `latest`, any `/specification/latest/basic/lifecycle...` and `/specification/latest/basic/transports#...` links (and anchors like `#session-management`, `#version-negotiation`, `#capability-negotiation`, `#setting-log-level`) will break; nav entries and redirects for the deleted/split/added pages must be reviewed alongside the per-page fixes below.
